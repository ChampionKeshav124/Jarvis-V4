# JARVIS V4 - Multilingual AI Desktop Assistant
> A step closer to building a real JARVIS.

JARVIS V4 is an upgraded AI desktop assistant built using Electron and Python, now enhanced with multilingual intelligence and wake-word activation.

This version allows JARVIS to understand and respond in multiple languages, making the assistant feel more natural, global, and interactive.

## 🚀 Features

- 🧠 AI-powered responses using Gemini API  
- 🌍 Multilingual support (English, Hindi, Spanish, French)  
- 🎤 Wake word activation ("Jarvis wake up")  
- 🔊 Voice output using ElevenLabs  
- 🖥️ Electron-based desktop application  
- ⚡ Real-time response system  
- 🎛️ Voice ON/OFF toggle  
- 📊 Status indicators (Idle, Listening, Processing)

## 🧪 Example Interaction

User: "Jarvis wake up"  
JARVIS: "I'm here. How can I assist?"

User: "नमस्ते"  
JARVIS: "नमस्ते, मैं आपकी कैसे सहायता कर सकता हूँ?"

User: "Hola"  
JARVIS: "Hola, ¿en qué puedo ayudarte?"

## 🧱 Tech Stack
